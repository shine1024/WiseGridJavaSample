기본예제
엑셀수식
엑셀과 같이 수식을 이용한 기능 구현이 가능합니다. (일부 수식만 지원)
다른점은 구분자를 엑셀은 "," WiseGrid는 ";"를 사용합니다.
(※엑셀 수식이 적용된 셀의 개수가 증가할 수록 비례해서 전체 연산 속도가 증가할 수 있습니다.)
SUM 첫번째 인자로 2번째 인자의 영역과 비교하여 컬럼인덱스(3번째 인자)에 해당하는 값을 출력합니다.
ex) =vlookup(B5;E1:F4;2;false)
※WiseGrid영역의 밑줄이 표시된 셀은 엑셀 수식이 적용된 셀입니다. 해당 셀을 클릭하면 적용된 수식을 확인할 수 있습니다.
